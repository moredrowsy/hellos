# Go Hello World Example

## Run from source

```bash
go run hello.go
```

Pass arguments

```bash
go run hello.go your name
```

## Build

This will compile the 'hello' program into the $GOPATH/bin folder

```bash
go install github.com/moredrowsy/hellos/go/hello
```

## Run from binary

```bash
hello
```

## Find location of 'hello' binary

```bash
which hello
```
